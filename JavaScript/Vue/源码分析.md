# 源码分析

在解析源码之前，需要先了解几个技术点

1. `[].slice.call(obj)`：将伪数组转化为真数组，所谓伪数组，其实就是一个对象，而这个对象拥有`length`属性，而且我们可以通过下标的方式去找到某个属性。

2. `node.nodeType`：节点类型

3. `Object.defineProperty(obj, propertyName, {})`：给对象添加属性，后面的对象是属性描述符

   属性描述符里面有两个比较重要的函数：

   ```javascript
   {
       // 属性在内存中的值
       value: ''
       // 是否可以重新定义
       configurable: true,
       // 是否可以枚举
       enumerable: true,
       // 是否可以重写属性值
       writable: true,
       // 可以动态某个属性的value
       get: function () {
           // ...
       }
       // 监听某个属性value值的的变化
       set: function () {
           // ...
       }
   }
   ```

### JSX实现

我们随便写一个jsx，再使用babel转义，比如:

```html
<div id="item">test</div>

<div id="item">
    <span></span>
</div>
```

转化为

```javascript
React.createElement('div', null);
// li 转化
React.createElement('li',{id: 'item'}, 'test');
```

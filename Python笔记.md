注释

不要描述代码，而是去解释这段代码的目的，python文件的命名一定不能有大写字母，也不能以数字开头，严格缩进

单行注释：#开头

多行注释：

```python
"""
我是一条注释
"""
```

语句以换行符结束，不要加分号

## 变量

不需要声明，直接进行赋值，并使用，但是必须赋值后使用

```python
# 变量的命名规范
# 所有字母小写
person_name = '张三'
print(person_name)
# 帕斯卡命名法
PersonName = '李四'
# 二进制
c = 0b10
# 八进制
d = 0o10
# 十六进制
f = 0x16
# 字符串,单引号和双引号都可以
string_one = 'hello'
string_two = "world"
```
